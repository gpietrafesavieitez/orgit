<b>Nombre:</b> Orgit<br>
<b>Versión:</b> 1.0.0<br>

<br>

$ bash orgit-1.0.0.sh<br>

<b>Uso en entornos Windows:</b><br>
Ejecutar => orgit-1.0.0.bat
